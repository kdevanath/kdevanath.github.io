Summary: Latitude Vs. X

The purpose of this project was to compare how weather changes as you get closer to the equator. To accomplish this we randomly generated data from OpenWeather.org using their api, for about 500 cities.


After assembling the dataset, used matplotlib to plot various aspects of weather vs. latitude. Factors included cloudiness, temperature, humidity, wind speed and humidity. This site provides the source data and visualizations created as part of analysis. Gives an explanation of trends and correlations that was observed.
